Simple physics simulation that features colisions, gravity, multiple objects interacting with each other inside a circle.
Developed in C#.
If you want to run the project in local you can simply clone it and execute the commmand "dotnet run" inside the folder


##Requisites

git

dotnet



##To execute:

git clone https://github.com/YoanAsghar/PhysicsSimulator

cd PhysicsSimulator

dotnet run
